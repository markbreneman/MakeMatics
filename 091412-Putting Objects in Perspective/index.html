SO WHAT'S IT ABOUT?

This paper deals with object detection in a 2D image.
The authors provide a framework for for object recognition based around three main parameters:


1. Objects in the image

2. Surface orientations
3. The camera viewpoint


Using these three parameters together, iteratively, they have a way of performing fairly accurate object detection.

Awesome...but whats the catch?

The research assumes:

  1.  All objects rest on the ground and are perpendicular to it.
  2.  The camera is not tilted or it has been corrected.
  3.  Object independence.  Overlapped objects are more difficult to identify.
  4.  Current object detection relies on a set of input training data.
